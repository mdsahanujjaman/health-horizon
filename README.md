***# ✨ Overview***



***Health Horizon is a next-generation AI-driven healthcare platform designed to modernize patient care, doctor interaction, diagnostics, emergency response, and digital medical management.***



***The platform integrates:***



***- 🤖 AI Diagnostics***

***- 👨‍⚕️ Smart Doctor Consultation***

***- 💬 Real-Time Communication***

***- 💊 Digital Prescriptions***

***- 🧠 Mental Health Assistance***

***# 🚀 Quick Start***



***## Clone Repository***



***```bash***

***git clone https://github.com/mdsahanujjaman/health-horizon.git***

***```***



***---***



***## Frontend Setup***



***```bash***

***npm install***

***npm run dev***

***```***



***---***



***## Backend Setup***



***```bash***

***cd health-horizon***

***npm install***

***npm start***

***```***
